ESP8266 door bell software to Key-module and Gong-module
Module is manged via a mqtt server

please use Arduino Tools to build ESP32-01 (esp8266) image

libraries:
- ESP8266 board generic version 2.5.2
  https://github.com/esp8266/Arduino

- PubSubClient version 2.6.0 !! by Nick O'Leary
- OneWire version 2.3.4 by Jim Studt and other
